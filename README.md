# 📊 Unemployment Analysis in India using Python

## 📌 Project Overview

This project analyzes unemployment trends in India using Python. The analysis focuses on understanding unemployment patterns before and during the COVID-19 pandemic through data cleaning, exploratory data analysis (EDA), and data visualization.

The project was completed as part of the **CodeAlpha Data Science Internship**.

---

## 🎯 Objectives

- Analyze unemployment trends across different states in India.
- Compare unemployment rates between Urban and Rural areas.
- Study the impact of COVID-19 on unemployment.
- Identify states with the highest unemployment rates.
- Explore employment and labour participation trends.
- Visualize the data using informative charts and graphs.

---

## 📂 Dataset Information

The project uses two unemployment datasets containing information such as:

- Region (State)
- Date
- Area (Urban/Rural)
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

---

## 🔍 Key Insights

- Most unemployment rates were concentrated at lower values with a few high outliers.
- Urban areas generally experienced higher unemployment than rural areas.
- A significant increase in unemployment was observed after the COVID-19 lockdown.
- Employment levels declined during the pandemic before gradually recovering.
- Labour participation showed noticeable changes over the study period.
- States such as Tripura and Haryana recorded relatively higher average unemployment rates.
- Correlation analysis showed a weak negative relationship between unemployment and employment.
